# YouTube TUI

A terminal user interface for YouTube, built with Go.

## Prerequisites

This project requires `yt-dlp` to be installed on your system.

### Installing yt-dlp

**On Linux (Ubuntu/Debian):**
```bash
sudo apt install yt-dlp
```

**On macOS:**
```bash
brew install yt-dlp
```

**On Windows:**
```bash
choco install yt-dlp
```

Or install via pip:
```bash
pip install yt-dlp
```

For more installation methods, visit [yt-dlp's official GitHub repository](https://github.com/yt-dlp/yt-dlp).

## Building

```bash
go build
```

## Running

```bash
./youtube-tui
```
